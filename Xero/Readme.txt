Xero Integration

To send invoice to users and keep track of invoice and payments information on Xero.

Purpose:
After performing inspection of property, the PCS Team member sends the invoice to the property owner.

Technical Implementation :
 - Get Access Token
 - Refresh Access Token
 - Create Xero Contact ID of Users
 - Send Invoice
 - Sync Xero invoices with the PCS appliction
 - Get Status of invoice
